Welcome to pattern_analysis's documentation!
=============================================

Welcome to the pattern_analysis package which is designed to empower efficient and thorough processing of biological sequencing data. This package offers a versatile toolkit for extracting meaningful patterns and insights from the sequencing datasets.

Current features:
Quantify pattern occurrences

Check out the :doc:`usage` section for further information.
